윈도우에서 링크를 눌렀을 때 브라우저가 실행되는데<br>
크롬 브라우저에 경우 시크릿모드로 열리거나, 특정 계정에서 열리거나 할 수 있다.

레지스트리를 수정하면 되는데 경로는 다음과 같다.

**"HKEY_CLASSES_ROOT\ChromeHTML\shell\open\command"**

기본값으로 이렇게 되어있을 것이다.<br>
"C:\Program Files\Google\Chrome\Application\chrome.exe" --single-argument %1

여기서 파라미터(옵션)를 추가해주면 된다.

먼저, 시크릿모드로 열리게 하려면 ***--incognito*** 해당 파라미터를 넣어주면 된다.

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --incognito --single-argument %1
```
***

다음으로 특정 계정에서 열리게 하기 위해서는 ***--profile-directory*** 파라미터와 ***Profile 1*** 인자를 넣어주면 된다.

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --profile-directory="Profile 3" --single-argument %1
```

두 개 다 넣을 경우 예시 

```
"C:\Program Files\Google\Chrome\Application\chrome.exe" --profile-directory="Profile 3" --incognito --single-argument %1
```
